NuSmartyPants
=============

NuSmartyPants is an implementation of [SmartyPants][sp] in [Nu][nu].

[sp]: http://daringfireball.net/projects/smartypants
[nu]: http://programming.nu

NuSmartyPants may be built as a framework using `nuke`.
